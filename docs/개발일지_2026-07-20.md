# 개발일지 — 2026-07-20

## 금일 작업

- Langfuse를 연결하여 Agent별 실행시간·토큰·비용을 확인할 수 있도록 하였다.
- SSE 기반 진행률과 예상 남은 시간(ETA) 표시 기능을 구현하였다.
- API 키 없이도 전체 파이프라인을 점검할 수 있는 더미 모드를 활용해 동작을 검증하였다.
- 프로젝트명만 입력하면 설명·목표 사용자·문제·키워드를 생성하는 AI 입력 자동완성 기능을 추가하였다.
- 분석 결과를 항목별 탭으로 구분하여 결과 화면의 가독성을 개선하였다.
- LLM 호출 실패 시 `[더미]` 대신 원인별 안내(혼잡/연결 등)를 표시하고, 근거 없는 통계(환각)를 탐지·경고하는 기능을 추가하였다.

## 누적 개발 현황

- 아이디어 입력부터 시장조사, 고객·경쟁·환경 분석, 기획서 작성까지 자동화하였다.
- Tavily 웹 검색을 통해 관련 자료와 출처 URL을 수집하고 기획서에 반영하도록 구현하였다.
- 초안 자동 평가와 재작성, 최종본 재평가 및 근거 일치성 검토 기능을 구축하였다.
- 사용자 수정 요청과 실행 이력 저장 기능을 구현했으며, 결과는 Word·Markdown·JSON 형식으로 내보낼 수 있다.
- 또한 LLM 호출 실패와 근거가 확인되지 않은 주장을 사용자에게 원인 및 경고 형태로 표시하도록 하였다.
