### 2) 이 프로젝트의 핵심 메시지 (보고서 전체를 관통시킬 한 줄)

> **"Multi-Agent의 가치는 '점수'가 아니라 '추적 가능한 근거'에 있다."**
> 단일 LLM은 출처 URL 0건, 근거 기반 Multi-Agent 파이프라인은 실제 출처 5건을 인용한다 — 모델을 바꿔도 뒤집히지 않는 **구조적 차이**. 그리고 우리는 그 검증의 **한계까지 정직하게 표시**한다.

## ⚠️ 정직성 원칙 (보고서에서 과장하지 말 것)

이 프로젝트는 "정직한 범위 표기"를 차별점으로 삼습니다. 보고서에서도 아래를 지켜 신뢰도를 높이세요.

- **출처 URL 수**는 "포함 개수"이며 URL 원문의 사실성까지 검증한 값이 **아님**을 명시.
- **비교 실험**은 "검색 없는 단일 프롬프트 vs 웹검색 포함 파이프라인"으로, 분업 효과와 검색 효과가 **섞여** 있음을 명시.
- **다중 모델 비교(3주제)**는 일반화가 아니라 **방향성 확인**으로 표기.
- **근거 검증**은 "수집된 검색 요약과의 일치"이지 URL 원문 사실 검증이 아님.
